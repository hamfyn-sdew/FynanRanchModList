### requirements

## purpose
the purpose of this mod collection is to create a version of the game for us to enjoy multiplayer with:
- QOL additions to tweak some more 'grind-y' aspects of the game
- customizable gameplay tweaks of certain mechanics
- adding additional content for artisan goods, farming, animals, and more
- adding more social gameplay to enhance NPC relationship & marriage system
## instructions
- install stardew on steam
- install SMAPI ([linux instructions](https://stardewvalleywiki.com/Modding:Installing_SMAPI_on_Linux))
